## 2. 图片规格说明

### 2.1 通用格式要求

| 格式 | 推荐用途 | 说明 |
|------|----------|------|
| **JPG/JPEG** | 照片类图片（产品、工厂）| 质量设置 80-85%，平衡质量与体积 |
| **PNG** | Logo、含透明背景的图片 | 无损格式，体积较大 |
| **WebP** | 优化版本（可选） | 体积比JPG小30%，现代浏览器支持 |
| **SVG** | 图标、Logo矢量版 | 无限缩放不失真 |

### 2.2 禁止的图片格式

- ❌ BMP（体积过大）
- ❌ TIFF（浏览器不支持）
- ❌ PSD（设计源文件）
- ❌ RAW（相机原始文件）

### 2.3 文件大小限制

| 图片类型 | 建议大小 | 最大不超过 |
|----------|----------|-----------|
| Hero 横幅大图 | 300-500KB | 800KB |
| 产品图片 | 150-300KB | 500KB |
| 证书/文件扫描 | 200-400KB | 600KB |
| 团队头像 | 50-100KB | 200KB |
| Logo（PNG） | 30-80KB | 200KB |

> 💡 如果原图过大，可使用以下工具压缩：
> - 在线工具：[TinyPNG](https://tinypng.com)、[Squoosh](https://squoosh.app)
> - 本地工具：ImageOptim（Mac）、FileOptimizer（Windows）

## 4. 拍摄建议

### 4.1 工厂环境拍摄

**推荐时间：**
- 白天生产高峰期（有人工作，画面有活力）
- 早晨开工前（整洁有序，适合全景）

**拍摄设备：**
- 单反相机或高端手机（iPhone Pro / 华为 P60 以上）
- 广角镜头（工厂内部拍摄）
- 三脚架（长曝光/夜间拍摄）

**拍摄角度：**
```
推荐角度：
✅ 45度斜侧方，体现纵深感
✅ 低角度仰拍，体现设备高大
✅ 高处俯拍（如操作平台），体现规模
✅ 炉前拍摄，捕捉金属熔液发光效果

不推荐角度：
❌ 正面平拍（缺乏立体感）
❌ 逆光（人物剪影，细节丢失）
❌ 过度堆砌设备（显得凌乱）
```

**建议拍摄清单：**

```
工厂外观：
□ 正门/大门（白天）
□ 厂区全景（无人机鸟瞰，可选）
□ 厂房外立面

生产区域：
□ 铸造生产线全景（至少2-3张）
□ 熔炉/冲天炉（安全距离外，捕捉光线）
□ 浇注过程（动态感，有金属液流）
□ 造型区域（模具和砂型）
□ 清理区域（机器人或人工操作）

检测区域：
□ 三坐标测量仪操作场景
□ 光谱分析仪
□ 超声波探伤仪
□ 产品检验台

产品区域：
□ 各类产品堆放/展示区
□ 单品特写（多角度）
□ 与参照物对比（如人手或尺子）
```

### 4.2 产品拍摄

**拍摄环境：**
- 建议租用专业摄影棚或使用白色背景纸
- 使用摄影灯箱（小型铸件）
- 避免杂乱背景

**产品主图标准（电商级）：**
```
背景：纯白色 RGB(255,255,255) 或浅灰色 RGB(240,240,240)
光线：均匀柔光，无强烈反光或阴影
角度：正视图 + 45度侧视图（至少2张）
产品：清洁干净，无油污、锈斑、毛刺
分辨率：≥ 2000 × 2000px（主图），可缩小使用
```

### 4.3 团队照片

**专业商务头像标准：**
```
背景：纯白、浅灰或浅蓝色背景
服装：正装（西装领带）或工作服（有工牌）
表情：自然微笑，目视镜头
光线：正面柔光，无过曝或欠曝
拍摄距离：胸部以上，面部约占画面1/2
```

## 5. 图片处理规范

### 5.1 图片压缩

```bash
# 使用 ImageMagick 批量压缩（命令行）
mogrify -quality 85 -resize 1920x1080> *.jpg

# 使用 ffmpeg 转 WebP
ffmpeg -i input.jpg -quality 85 output.webp
```

### 5.2 图片命名规范

```
格式：{类别}-{描述}-{序号}.{扩展名}

示例：
✅ hero-banner.jpg
✅ product-grey-iron-casting-01.jpg
✅ team-zhang-jianguo.jpg
✅ cert-iso9001.jpg

禁止：
❌ IMG_20240115_123456.jpg   （相机原始名）
❌ 图片1.jpg                  （中文命名）
❌ product photo.jpg          （含空格）
❌ CASTING_PRODUCT_FINAL_V3_NEW_OK.jpg  （过长）
```

## 7. 常见问题

**Q：我只有手机拍的照片，可以用吗？**  
A：现代旗舰手机（iPhone 15 Pro、华为 P60 Pro等）拍摄质量完全可以用于网站。关键是保证：构图合理、光线充足、拍摄清晰、后期适当调色。

**Q：没有专业摄影棚，产品图怎么拍？**  
A：可以用白色大纸张铺在桌面作为背景，靠近窗户借助自然光，避免正午直射光（会有硬阴影）。在家得宝或宜家购买两盏台灯（磨砂白色灯泡）打柔光效果也不错。

**Q：图片太大上传很慢怎么办？**  
A：使用 [TinyPNG](https://tinypng.com) 在线压缩，或 [Squoosh](https://squoosh.app) 压缩并转换为 WebP 格式，可减小 40-60% 的文件体积。

**Q：临时没有真实照片，能先上线吗？**  
A：可以暂时使用免费图库图片（[Unsplash](https://unsplash.com)、[Pexels](https://pexels.com)），但需在 1-2 个月内替换为真实照片，真实照片对客户转化率有显著提升效果。

**Q：产品图需要去背景（抠图）吗？**  
A：电商主图建议去背景，官网氛围图不必须。如需抠图，可使用 [remove.bg](https://remove.bg) 自动完成。

**Q：需要准备多少张图片才能上线？**  
A：最低要求：Hero图1张 + 工厂图1张 + 产品封面图6张 = 共8张。建议准备本文档中标注"最高"和"高"优先级的所有素材（约20张）再正式上线。
