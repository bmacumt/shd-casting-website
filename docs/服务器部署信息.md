## 部署新项目注意事项

1. **阿里云安全组**：新端口需在阿里云控制台 → 安全组中放行
2. **磁盘空间**：剩余 31 GB，注意 Docker 镜像体积
3. **内存**：3.4 GB 总量，当前 courseagent 占用约 1.3 GB，剩余约 2 GB 可用
4. **Docker 离线部署**：本地 `docker save` 导出 tar → SCP 上传 → `docker load` 导入
5. **`.env` 修改后**：必须 `docker compose up -d --force-recreate` 而非 `docker compose restart`
6. **前端 dist 覆盖**：删除旧文件后用 `scp dist/* server:/path/` 上传（不要用 `scp -r dist/ server:/path/`，会嵌套目录）
