## 1. 项目概述

### 1.1 项目背景

本项目为上海铸造有限公司（SHD Casting Co., Ltd）官方网站，主要面向国内外采购商、工程师和贸易商，展示公司实力、产品目录，并提供在线询价功能。

### 1.2 核心目标

| 目标 | 说明 |
|------|------|
| 品牌展示 | 展示公司20年行业经验、生产能力和资质认证 |
| 产品推广 | 清晰展示各类铸件产品规格和应用场景 |
| 询价转化 | 引导访客填写询价表单，提高销售线索转化率 |
| 搜索优化 | 通过SEO吸引国内外精准客户 |
| 多语言 | 支持中文/英文切换（二期目标） |

### 1.3 用户角色

| 角色 | 描述 |
|------|------|
| **访客** | 普通网站访客，浏览产品和公司信息 |
| **询价客户** | 填写询价表单的潜在客户 |
| **管理员** | 公司内部人员，管理产品和处理询价 |

## 3. 图片资源说明

### 3.1 是否必须提供图片？

**是的，必须替换**。当前前端代码中的图片均来自 Unsplash（免费图库），仅用于开发预览。正式上线前必须替换为**公司真实产品和工厂照片**，原因：

1. Unsplash 图片为通用图，不代表公司真实产品
2. 真实图片能大幅提升客户信任度和转化率
3. 避免图片版权或链接失效问题

### 3.2 图片存放位置（两种方案）

#### 方案A：本地静态图片（简单，适合小型网站）

将图片放在 `frontend/public/images/` 目录下：

```
frontend/public/images/
├── hero/
│   └── hero-banner.jpg          (首页横幅大图)
├── about/
│   ├── factory-overview.jpg     (工厂全景)
│   └── production-line.jpg      (生产线)
├── products/
│   ├── grey-iron-casting.jpg    (灰铸铁件)
│   ├── ductile-iron-casting.jpg (球墨铸铁件)
│   ├── steel-casting.jpg        (铸钢件)
│   ├── aluminum-casting.jpg     (铝合金铸件)
│   └── precision-casting.jpg   (精密铸件)
└── certifications/
    ├── iso9001.jpg
    └── ce-cert.jpg
```

在代码中引用方式：
```tsx
<img src="/images/hero/hero-banner.jpg" alt="铸造工厂" />
```

#### 方案B：云存储 CDN（推荐，适合正式上线）

使用阿里云 OSS 或腾讯云 COS：

```
https://your-bucket.oss-cn-shanghai.aliyuncs.com/images/hero/hero-banner.jpg
```

优点：加载速度快、全球 CDN 加速、不占用服务器带宽

### 3.3 代码中的图片替换方法

打开对应页面文件，找到图片 URL 常量，替换为你的图片路径：

```tsx
// 替换前（HomePage.tsx 第1行）
const heroImg = "https://images.unsplash.com/photo-xxxx...";

// 替换后（本地图片）
const heroImg = "/images/hero/hero-banner.jpg";

// 替换后（云存储）
const heroImg = "https://your-cdn.com/images/hero/hero-banner.jpg";
```

详细图片规格请查阅：[素材说明.md](./素材说明.md)

## 5. 后端功能需求

### 5.1 询价管理模块（核心）

**功能描述：**
- 接收前端提交的询价表单数据
- 将数据存入数据库
- 自动发送邮件通知给销售团队
- 自动发送确认邮件给客户
- 管理员可在后台查看、处理、回复询价

**数据字段：**

| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | INT / UUID | 主键 |
| name | VARCHAR(100) | 客户姓名 |
| company | VARCHAR(200) | 公司名称 |
| email | VARCHAR(200) | 邮箱（必填） |
| phone | VARCHAR(50) | 电话 |
| product_category | VARCHAR(100) | 产品类别 |
| quantity | VARCHAR(100) | 预计采购量 |
| message | TEXT | 详细需求 |
| status | ENUM | 状态：pending/processing/replied/closed |
| source | VARCHAR(50) | 来源：website/wechat/phone |
| ip_address | VARCHAR(50) | 访客IP（防刷） |
| created_at | DATETIME | 创建时间 |
| updated_at | DATETIME | 更新时间 |
| replied_at | DATETIME | 回复时间 |
| admin_note | TEXT | 内部备注 |

### 5.2 产品管理模块

**功能描述：**
- 后台增删改查产品信息
- 支持产品分类管理
- 产品图片上传（对接云存储）
- 产品排序和上下架控制

**产品数据字段：**

| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | INT | 主键 |
| name | VARCHAR(200) | 产品名称 |
| category_id | INT | 分类ID（外键） |
| material | VARCHAR(200) | 材质规格 |
| weight_range | VARCHAR(100) | 重量范围 |
| standard | VARCHAR(100) | 执行标准 |
| description | TEXT | 产品描述 |
| cover_image | VARCHAR(500) | 封面图URL |
| images | JSON | 多图URL数组 |
| tag | VARCHAR(50) | 标签（热销/新品/精品） |
| is_featured | BOOLEAN | 是否首页推荐 |
| is_active | BOOLEAN | 是否上架 |
| sort_order | INT | 排序权重 |
| created_at | DATETIME | 创建时间 |

### 5.3 内容管理模块（可选，CMS）

用于管理静态内容（公司简介、团队介绍、发展历程等），避免每次修改都需要改代码。

### 5.4 邮件通知需求

| 场景 | 接收方 | 邮件内容 |
|------|--------|----------|
| 新询价提交 | 销售团队 | 客户信息 + 询价内容 |

## 7. SEO与性能要求

### 7.1 SEO基础要求

- 每个页面配置独立的 `<title>` 和 `<meta description>`
- 图片添加 `alt` 属性（含关键词）
- 页面URL使用语义化路径（已实现）
- 生成 `sitemap.xml`
- 配置 `robots.txt`
- 页面加载时间 < 3秒（首屏）

### 7.2 关键SEO词

```
中文关键词：铸件厂家、灰铸铁件、球墨铸铁件、铸钢件、上海铸造
英文关键词：casting manufacturer, iron casting, steel casting, Shanghai foundry
```

### 7.3 性能目标

| 指标 | 目标值 |
|------|--------|
| 首屏加载时间 | < 2.5秒 |
| Lighthouse Performance | > 85分 |
| 图片格式 | WebP（附加JPG降级） |
| 图片懒加载 | 必须实现 |

## 8. 部署说明

### 8.1 前端部署

```bash
# 构建生产版本
cd frontend
pnpm build

# 产物在 dist/ 目录
# 使用 Nginx 托管静态文件
```

### 8.2 Nginx 配置示例

```nginx
server {
    listen 80;
    server_name www.shdcasting.com;
    root /var/www/dist;
    index index.html;

    # 前端路由支持
    location / {
        try_files $uri $uri/ /index.html;
    }

    # API 反向代理
    location /api/ {
        proxy_pass http://localhost:3000/;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }

    # 图片缓存
    location /images/ {
        expires 30d;
        add_header Cache-Control "public, immutable";
    }
}
```

### 8.3 环境变量配置

前端需要在 `.env.production` 中配置：

```env
VITE_API_BASE_URL=https://api.shdcasting.com
VITE_SITE_NAME=上海铸造有限公司
```
